# apps/

Container directory for Django applications specific to IndusMind AI
(for example: landing page, chatbot UI, dashboard, document management).

**Status:** Empty foundation placeholder. No Django app has been created
yet. Apps will be added here as individual modules (e.g. `apps/landing`,
`apps/chatbot`) in future development phases, and registered in
`backend/config/settings.py` under `LOCAL_APPS` at that time.
